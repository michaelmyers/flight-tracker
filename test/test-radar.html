Technical Details

The mini radar implementation includes:

  • Frontend: Canvas-based radar visualization with WebSocket client
  • Backend: WebSocket server broadcasting aircraft data every 2 seconds
  • Data Source: PiAware dump1090 JSON feed
  • Update Rate: 2-second intervals for smooth tracking
